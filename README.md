**Research engineer** focused on large‑model inference, low‑level GPU compatibility, and edge AI.  
Specialized in speculative decoding, MoE routing, and multi‑token‑parallel (MTP) pipelines.  
Creator of self‑hosted AI stacks that run on consumer and datacenter GPUs not officially supported by CUDA.

| **GPU Tech** | CUDA sm_61 shim, TensorRT, NVML, RTX‑30xx, Tesla P40 |

### LLM Research
- [gguf-per-cohere-30b-a3b-su-p40](https://github.com/chessarisilvio/gguf-per-cohere-30b-a3b-su-p40) – Ported Cohere 30B A3B to Tesla P40 using a CUDA sm_61 compatible fork of llama.cpp.  
- [add-video-input-support-to-llamacpp-mtmd](https://github.com/chessarisilvio/add-video-input-support-to-llamacpp-mtmd) – Integrated real‑time video frame streaming into llama.cpp’s multi‑token‑parallel mode.  
- [automated-exl2-conversion-validation-pipeline](https://github.com/chessarisilvio/automated-exl2-conversion-validation-pipeline) – End‑to‑end toolchain for EXL2 quantisation, mixed‑precision conversion, and benchmark validation of Qwen‑derived models.  
- [mtop-htop-per-monitoraggio-llm-locale](https://github.com/chessarisilvio/mtop-htop-per-monitoraggio-llm-locale) – htop‑style UI that reports GPU utilization, VRAM pressure, temperature, and token‑generation rate for locally hosted LLMs.

### AI Infrastructure
- [openclaw](https://github.com/chessarisilvio/openclaw) – Node.js gateway that proxies Ollama requests to locally compiled LLM servers, handling model routing and auth.  
- [git-mcp-server-per-agenti-ai](https://github.com/chessarisilvio/git-mcp-server-per-agenti-ai) – FastAPI‑based MCP (Model Context Protocol) server exposing Git operations via JSON for autonomous AI agents.  
- [cuda-sm61-compatibility-shim](https://github.com/chessarisilvio/cuda-sm61-compatibility-shim) – Compatibility layer that maps CUDA sm_61 kernels to Pascal GPUs (Tesla P40), enabling modern LLM inference on legacy hardware.  

### Tools & Automation
- [megatoool](https://github.com/chessarisilvio/megatoool) – Full‑stack OSINT suite (C++ core, Flask web UI, AI‑powered image analysis).  
- [bot-short](https://github.com/chessarisilvio/bot-short) – Telegram bot in C++ that generates vector graphics via local LLM and returns SVG/EPS files.  
- [ai-dashboard](https://github.com/chessarisilvio/ai-dashboard) – Local web dashboard (port 9190) aggregating GPU stats, service health, and an agenda generated by a 35B model.  
- [reddit-monitor](https://github.com/chessarisilvio/reddit-monitor) – Scheduler that scrapes AI‑focused subreddits, extracts topics, and stores them in a searchable markdown agenda.  

### Embedded & Hardware
- [controller-termico-proattivo-esp32](https://github.com/chessarisilvio/controller-termico-proattivo-esp32) – ESP32 firmware that reads dual DS18B20 sensors on a Tesla P40 and drives a MOSFET‑based fan PWM loop for proactive thermal management.  
- [digital-thermal-lcd](https://github.com/chessarisilvio/digital-thermal-lcd) – USB‑HID driver and Python service that pushes real‑time CPU/GPU temperatures to a Thermalright LCD module.  
- [ai-model-selector-physical-controller](https://github.com/chessarisilvio/ai-model-selector-physical-controller) – ESP32‑based rotary encoder and OLED UI for selecting locally hosted AI models, communicating via HID or OpenClaw gateway.
